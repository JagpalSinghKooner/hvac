## How to Use This Roadmap

### Starting a New Phase

1. Open the phase file (e.g., `phase-6-branding-conversion.md`)
2. Read the **Status** section to verify prerequisites are complete
3. Copy the **Starting Prompt** (the code block) and use it to begin the phase
4. Follow the **Component Checklist** to track progress
5. Reference the **Expected Outcomes** to know when the phase is complete

### During a Phase

- Use the component checklists to track what's built
- Refer to the focus areas for guidance
- Mark items as complete as you build them
- Add notes about decisions made or issues encountered

### Completing a Phase

- Verify all items in the checklist are complete
- Ensure all expected outcomes are achieved
- Update the roadmap README.md to mark phase as complete
- Move to the next phase

## Key Principles

> Each phase should reduce ambiguity, not introduce it.
> Progress is measured by **stability and clarity**, not speed.

### Phase 6 Specific Principles

1. **Lead Generation First**: Every component should support conversion goals
2. **Mobile-First**: Most HVAC searches are on mobile devices
3. **Trust Signals**: Local customers need credibility indicators
4. **Schema Early**: Add structured data as you build components, don't wait for Phase 7
5. **Professional Polish**: This is when the site looks like a real business
